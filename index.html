PBR Texture Mapping Guide & Info

본 도구는 3D 그래픽스 및 게임 개발 과정에서 자주 사용되는 유니티(Unity URP) 텍스처 포맷과 언리얼 엔진(Unreal Engine ORM) 텍스처 포맷을 클라이언트 측에서 안전하고 빠르게 상호 변환해 주는 무료 PBR 텍스처 컨버터입니다.

1. Unity URP ↔ Unreal ORM 채널 매핑 구조

  • Unity URP (Metallic Smoothness): Red 채널에 Metallic 값이 저장되며, Alpha 채널에 Smoothness(매끄러움) 값이 저장됩니다.
  • Unreal Engine ORM: Red 채널에 Ambient Occlusion(차폐), Green 채널에 Roughness(거칠기 = 1 - Smoothness), Blue 채널에 Metallic(금속성) 값이 결합되어 하나의 맵으로 사용됩니다.

2. 주요 제공 기능

  • Unity → ORM: 메탈릭-스무스니스 맵을 언리얼 표준 Roughness 및 Metallic 구조로 채널을 재배치하여 내보냅니다.
  • ORM → Unity: 언리얼의 ORM 맵을 분석하여 유니티 엔진에 적합한 메탈릭-스무스니스 포맷으로 롤백합니다.
  • 채널 분리 및 RGBA 병합: 각 이미지의 R, G, B, A 단일 채널을 독립된 흑백 이미지로 분리하거나, 개별 흑백 텍스처를 하나의 완성된 RGBA 이미지 파일로 결합할 수 있습니다.

* 웹 브라우저 내에서 순수하게 자바스크립트로만 연산되므로, 사용자의 텍스처 리소스가 외부 서버로 절대 업로드되지 않아 유출 걱정 없이 안전하게 사용할 수 있습니다.
